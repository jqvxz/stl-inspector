
## stl-inspector - Intro

Easy-to-use web application that allows the user to open stl files and inspect them easily


## Installation 

- Download the html file
- Use any way to host it locally


## Authors

- [@jqvxz](https://github.com/jqvxz)


## Features

- Open stl files in the browser
- Fast user interface
- Change color of objects
- Measurements for files
- Multi object support
- Easy to use
